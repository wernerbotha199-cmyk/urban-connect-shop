# Urban Connect E-Commerce Platform

A fully functional e-commerce demo platform built with vanilla JavaScript, HTML, and CSS.

## Features

✅ **Complete E-Commerce Storefront**
- Browse 6 premium tech products
- Product filtering and search
- Detailed product pages
- Real-time price calculations

✅ **Shopping Experience**
- Add to cart functionality
- Shopping cart management
- Wishlist support
- Persistent cart (localStorage)

✅ **Checkout & Orders**
- Multi-step checkout process
- Order confirmation
- Order history tracking
- Order status management

✅ **Admin Dashboard**
- Real-time KPI metrics
- Order management
- Revenue tracking
- Customer analytics

✅ **AI Assistant**
- Emma-i™ AI chatbot
- Shopping recommendations
- Customer support

## Quick Start

Simply open `index.html` in a browser. No backend required for the demo.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to https://vercel.com/new
3. Import this GitHub repository
4. Select SA-iLabs team
5. Click Deploy
6. Get your live URL instantly!

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Storage:** LocalStorage (browser-based)
- **Design:** Responsive, Mobile-first
- **Styling:** Modern CSS with Tailwind-inspired utilities
